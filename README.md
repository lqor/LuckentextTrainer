# Luckentext Trainer
Der Lückentrainer ist ein Tool zum Üben von bestimmten Aufgaben(Lückentexte, wo ein Wort fehlt), die besonders viel Schwierigkeiten für Deutschlernende bereitstellen.

## Was wurde benutzt:
* Spring Boot
* Hibernate + JPA
* MySQL
* Spring Security
* JSP
